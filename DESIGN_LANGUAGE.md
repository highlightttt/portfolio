# Portfolio Design Language

## 四个关键词

### 动态 Dynamic / Emergent
没有预设的死规则和固定流程。行为从简单规则中**涌现**，没人预设结果。每次打开都不一样，每个人看到的都不一样。系统自己生成，不是设计师画死的。

**视觉语言：** 粒子聚合成形态、交互改变结构、生长动画、永远在运动

### 有机 Organic
像生命一样自然生长，自由自在。对环境有感知、会回应 — 你动它就变，你停它就静，像跟一个活物在对话。边界是模糊的，功能是弥漫的，你不需要找入口，它就在那里。没有直角和网格，一切像活的。

**视觉语言：** 曲线、呼吸感、没有硬边框、渐变过渡、背景即界面

### Primitive 基元
由最小单元组合出无限可能。原子→分子→生命。像 Notion 的 block、像 Claude Code 的 tool primitives — 不设计流程，设计基本能力块，让它们自由组合。

**视觉语言：** 粒子、最小单元、点。50万个基元 splat 粒子通过规则涌现出复杂形态

### 共生 Symbiotic
人和 AI 不是"使用者和工具"，是共生关系。不是"我用 AI 做了个东西给你看"，而是你、AI、观众三者同时在场，实时影响彼此。

三层共生：
- **创作者 ↔ AI** — 作品本身就是 Jesse + AI 共同完成的，不隐藏，甚至展示过程
- **作品 ↔ 观众** — 你看它的方式改变它的样子
- **观众 ↔ 观众** — 你的访问留下痕迹，下一个人看到的世界因你而不同

**视觉语言：** 两种粒子云互相渗透融合、鼠标轨迹生长出不可控形态、留痕系统

**比喻：** 珊瑚礁 — 珊瑚（结构/人）和虫黄藻（能量/AI）共生，单独拿走任何一方另一方都活不了
